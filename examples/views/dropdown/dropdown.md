## API

### Props

#### **Dropdown**
属性|类型|描述|默认值
---|---|---|---
value|Boolean|显示状态，支持`v-model`双向绑定|-
trigger|String|触发事件：`click`、`hover`|`click`
placement|String|显示位置：`bottom-start`、`bottom`、`bottom-end`、`top-start`、`top`、`top-end`|`bottom-start`
disabled|Boolean|设置组件禁用状态|-
